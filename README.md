# Sales Data Analysis
An e-commerce sales analysis project examining product performance, customer demographics, and business metrics.

# Project Overview
1. Dataset: 100 sales records with multiple dimensions
2. Analysis: 6 business intelligence questions
3. Tools: Python, Pandas, Matplotlib
4. Skills: Data analysis, business insights, correlation analysis

# Business Questions
1. Total sales by product category
2. Monthly sales trends
3. Top 5 most profitable products
4. Customer age distribution
5. Best performing region by sales
6. Correlation between sales, profit, and rating

# Technologies Used
1. Python 3
2. Pandas - Data manipulation
3. Matplotlib - Data visualization
